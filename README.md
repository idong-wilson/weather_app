# Weather App

A simple weather application that fetches and displays weather information for a specified city. The app allows users to search for a city, switch between Celsius and Fahrenheit units, and view detailed weather data including temperature, forecast, wind speed, humidity, pressure, sunrise, and sunset times.

## Features

* Search for weather information by city.
* Switch between Celsius and Fahrenheit units.
* Display detailed weather data including:
  * City and country name
  * Date and time
  * Weather forecast
  * Weather icon
  * Temperature (current, minimum, maximum, feels like)
  * Wind speed
  * Humidity
  * Pressure
  * Sunrise and sunset times

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* OpenWeatherMap API

## Setup and Installation

Prerequisites
* Node.js and npm installed on your machine.
* Tailwind CSS installed on your project. 

## Project Structure

* index.html: The main HTML file that contains the structure of the web page.
* styles.css: The CSS file that contains the styling of the web page. 
* index.js: The main JavaScript file that contains the logic for fetching and displaying weather data.
